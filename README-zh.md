# zzx-template
> 一个基于vue以及element的前端模板，参照vue-element-admin。
## 增加功能
1. 封装了常用的表单，图表等组件
2. 升级部分依赖版本
3. 增加element按需加载的功能
4. 增加全局共享scss变量的配置
5. 增加webpack gzip功能
## 项目启动
1. 安装依赖
```
npm install or cnpm install
```
2. 启动服务
```
npm run serve
```
## 项目构建
```
npm run build
```
## 其他script
`test:unit` : 单元测试  

`report` : 生成打包分析报告，在public下面的report.html  

`svgo` : 压缩指定文件夹下的svg

`lint` : 格式化代码
